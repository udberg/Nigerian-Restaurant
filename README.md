# Restaurant Page

![Microverse](https://img.shields.io/badge/-Microverse-6F23FF?style=for-the-badge)

## Description

This project was meant to enhance the knowledge on DOM manipulation by dynamically rendering a simple restaurant homepage from where I live. I have built it using Javascript & Webpack. This web page contains three tabs ( Home, Service & Visit-us).

In this project, I have created a web page of a local restaurant from where I live. I have built it using javascript & Webpack. This web page contains three tabs :

- A home page.
- A service page.
- A visit-us page.
